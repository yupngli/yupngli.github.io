中级SQL（1） -- CASE WHEN 与 子查询

发表于 2019-10-22 分类于 SQL

关于如何操纵，转换和充分利用数据的一系列教程，前提基础是充分掌握了SQL的聚合以及表联结。本中级教程包括一些必要的、关键函数，这些函数可以对关系数据库中的信息进行整理，过滤和分类，扩展SQL工具包并回答复杂的问题。 内容包括了：CASE语句的灵活用法、子查询。同时使用欧洲足球数据库发现有关足球的一些有趣事实。

巧用子查询（Subqueries）

基本概念

子查询可以是下面查询语句的一部分：

  • SELECT
  • FROM
  • WHERE
  • GROUP BY
