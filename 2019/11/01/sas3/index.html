用PROC步导入数据集

解释一下部分代码：

  • OUT option: Specify the new SAS data set name
  • DATAFILE option: Specify the full path to the data file like we did before with the INFILE statement
  • DBMS option: Simply the identifier of the file (比如CSV，xlsx)
  • GETNAMES = yes option: If the first row in your data file are the variable names that you want to use as SAS variable names（即：使不使用第一行作为列名）
  • DATAROW = ? : 从第几行开始读数据，比如如果第一行是列名，第二行空，第三行是数据，则DATAROW=3
  • SHEET : If you’re reading in data sets on multiple sheets of an Excel workbook, you can specify the name of the sheet explicitly and read in only that specific data set

**注意：若用 PROC IMPORT 读入没有变量名的数据文件，则variable names will default to var1, var2, var3, and so on.
