SAS可解决的任务的四个维度

SAS Studio的四个维度

  • Base SAS, which is the foundation of SAS software, a lot of built-in functions that you’ll need
  • SAS/ACCESS for reading in raw data files
  • SAS/STAT for doing some advanced analytic models
  • SAS/IML for doing everything in an interactive matrix language, and a few procedures from the ETS license, which is the time series and forecasting procedures.
